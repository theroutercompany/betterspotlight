BetterSpotlight File Search

BetterSpotlight is an offline-first file search application for macOS power users. It provides instant full-text search across your entire filesystem using SQLite FTS5 indexing.

Unlike traditional Spotlight, BetterSpotlight focuses exclusively on file search with configurable exclusion rules, multi-signal ranking, and support for over 100 file types.

System Architecture

The application uses a four-layer architecture with strict dependency direction: App, Core, Services, and Shared.

App Layer
SwiftUI/Qt interface with status bar integration
Core Layer
FS monitoring, content extraction, indexing, and ranking
Services Layer
Process-isolated indexer, extractor, and query services
Shared Layer
Codable models and IPC protocol definitions

Configuration

Exclusion patterns use gitignore-style syntax in .bsignore files. Default exclusions include node_modules, .git, build, and other common non-user directories.
