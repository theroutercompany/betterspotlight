# Literature Review

This review summarizes prior work on local semantic retrieval systems.
The methodology compares lexical-only and hybrid ranking approaches.
Several findings suggest that blended scores improve intent recall.

## Papers considered
- Lightweight vector indexing for desktop search
- Adaptive re-ranking with user feedback
- Privacy-preserving local language models

## Takeaways
Hybrid systems perform best when lexical precision remains dominant.
Evaluation must use stable fixtures and versioned corpora.
