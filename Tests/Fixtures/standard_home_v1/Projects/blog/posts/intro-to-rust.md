# Intro to Rust for App Developers

Rust helps teams build fast and safe systems software.
The ownership model prevents classes of memory bugs at compile time.
You will frequently use borrowing to pass references without copying data.
Understanding lifetime annotations can clarify API boundaries.

## Build and run
Use cargo build for compilation.
Use cargo test for verification.

## Why teams adopt Rust
- Strong type system
- Excellent tooling
- Predictable performance

## Practical tip
Start by porting isolated modules before rewriting large services.
